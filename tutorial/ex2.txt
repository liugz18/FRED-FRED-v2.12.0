FRED Exercises

2. CREATING A FRED PARAMETER FILE

Use a text editor to create a file called params that contains two
lines:

days = 100
fips = 42065

These parameters tell FRED to run a simulation of influenza epidemic in
Punxsutawney (Jefferson County), PA for 100 days.  All runtime
parameters not given in the params file will be given default values
specified in the file FRED/input_files/params.default.
